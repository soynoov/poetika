# PoetiKa

PoetiKa es una experiencia de escritura creativa con una estética editorial. Cada día propone tres palabras comunes para toda la comunidad y convierte esos textos en un feed público con identidad de autor.

## Qué puede hacer hoy

- Mostrar un reto diario con tres palabras compartidas.
- Permitir registro e inicio de sesión.
- Publicar relatos firmados dentro del reto activo.
- Dar likes y ordenar los textos según la respuesta de la comunidad.
- Mostrar perfiles públicos con bio, nombre visible y biblioteca personal.

## Experiencia de producto

PoetiKa prioriza una navegación simple, visualmente cuidada y centrada en la escritura. El tono general mezcla cuaderno, papel editorial y comunidad literaria ligera, evitando fricción innecesaria entre leer, escribir y volver al reto del día.

## Estado actual

El producto ya cubre el flujo principal: entrar, escribir, publicar, descubrir relatos y visitar perfiles. La base es válida para seguir iterando sobre comunidad, visibilidad de autores y profundidad de los retos sin rehacer la experiencia principal.

## Enfoque

PoetiKa no está planteada como un editor complejo, sino como un espacio directo para escribir con una consigna clara, publicar rápido y volver a leer cómo otras personas resolvieron el mismo desafío.
